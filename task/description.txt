Необходимо сверстать макет https://drive.google.com/drive/u/1/folders/13ll9vtqB4fu7m6hJy_Zc5gMp01KR7qgY 
и сделать анимации для секций по скролу. Варианты анимаций и адаптив на своё усмотрение.

Требования к вёрстке:

1. Не использовать Bootstrap и другие CSS фреймворки.
2. Использовать методологию БЭМ для именования классов.
3. Поддержка последних версий браузеров.
4. Для анимаций желательно использовать GSAP
5. Чем ближе к дизайну тем лучше.

Будет плюсом дополнительно сделать небольшую задачку: при наведении на блок с текстом буквы должны разлетаться в разные стороны,
 меняя свой размер и угол поворота.
